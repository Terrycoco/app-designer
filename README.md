# App Designer

A starter project for React, Redux & Webpack

###Also Includes###
* Testing with Mocha & Chai
* react-router
* palette designer
* Example Files


###Getting Started###


```
	> git clone git@github.com:terrycoco/app-designer.git <YourProjectName>
	> cd  <YourProjectName>
	> npm install
	> npm start
	> open a browser to localhost:8080
```
